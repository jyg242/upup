XHTML文件中class的命名

(1)颜色:使用色彩的名称或者16进制代码,如(不倡议以表示来命名)
　　.red { color: red; }
　　.f60 { color: #f60; }
　　.ff8600 { color: #ff8600; }
(2)字体大小，,直接应用"font+字体大小"作为名称,如
　　.font12px { font-size: 12px; }
　　.font9pt {font-size: 9pt; }
(3)对齐款式,使用对齐目的的英文名称,如
　　.left { float:left; }
　　.bottom { float:bottom; }
(4)标题栏样式，淘宝网女装冬装外套,使用"种别+功效"的方法命名,如
　　.barnews { }
　　.barproduct { }

其它相干留神事项

　　1.一律小写;
　　2.尽量用英文;
　　3.不加中杠跟下划线;
　　4.尽量不缩写，除非一看就清楚的单词.
